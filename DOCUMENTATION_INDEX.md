## 📞 Support

### If you encounter issues:
1. Check [START_HERE.md](START_HERE.md) troubleshooting section
2. Check [READY_TO_RUN.md](READY_TO_RUN.md) troubleshooting section
3. Check [VERIFICATION_CHECKLIST.md](VERIFICATION_CHECKLIST.md) for verification steps
4. Check browser console (F12) for errors

### Common Issues:
- Port in use → Change in vite.config.js
- Module not found → Run npm install
- Blank page → Clear cache
- Styles not loading → Restart dev server
- API errors → Check backend on port 5000
