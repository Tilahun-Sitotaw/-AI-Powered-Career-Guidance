  • Generic career advice doesn't match individual skills and interests
  • Lack of integrated platform for:
    • Learning path recommendations
    • Skill gap analysis
    • Internship opportunities
    • Scholarship information
    • Interview preparation
  • Need for AI-powered personalization at scale

AI-generated career roadmaps with phases, skills, and resources

Identifies missing skills for target roles with learning resources

Personalized internship opportunities based on skills and department

🏆 Scholarships

Relevant scholarship recommendations matching user profile

🎤 Interview Prep

AI-generated interview questions with suggested answers

💬 AI Chat

Real-time career guidance assistant

Key Features: Authentication

Secure User Management

  • Email Registration - Create account with email verification
  • OTP Login - One-Time Password sent via email
  • Password Hashing - Bcrypt for secure password storage
  • JWT Tokens - Secure API authentication

User Profile

  • Store skills, interests, department, year, preferred role
  • Profile-based personalization for all recommendations
  • Update profile anytime to regenerate recommendations

Key Features: AI Integration

Google Gemini API

  • Personalized Recommendations - Based on user profile
  • Model Fallback - Tries multiple models for reliability
  • Caching Strategy - Stores results in MongoDB
  • Profile Hash Detection - Regenerates when profile changes

Generated Content

  • Career paths with match scores
  • Learning roadmaps (3 phases)
  • Skill gaps with resources
  • Interview questions with answers
  • Scholarship recommendations
  • Internship opportunities

Database Schema

Collections

1. Users Collection

{ _id: ObjectId, name: String, email: String, password: String (hashed), skills: [String], interests: [String], department: String, preferredRole: String, createdAt: Date }

2. Recommendations Collection

Stores: careerPaths, roadmap, projects, skillGaps, interviewQuestions, scholarships, salaryInsights

API Endpoints

Authentication

POST /api/auth/register - Create account
POST /api/auth/login - Login with email
POST /api/auth/verify-otp - Verify OTP

Recommendations

GET /api/recommendations - Get all recommendations
POST /api/recommendations/regenerate - Force regenerate
POST /api/recommendations/roadmap - Get role-specific roadmap

Other Endpoints

GET /api/internships - Get internship opportunities
POST /api/chat/send - Send message to AI

Challenges & Solutions

⚠️ API Quota Limits

Solution: Model fallback, caching, profile hash detection

⚠️ Data Uniqueness

Solution: Enhanced Gemini prompts for personalization

⚠️ MongoDB Connection

Solution: IP whitelist (0.0.0.0/0), connection pooling

⚠️ Real-time Performance

Solution: Caching, async operations, indexing
