## 🚨 If You See Errors

### Error: "Port 5173 already in use"
- Close other apps using port 5173
- Or change port in `vite.config.js`

### Error: "Module not found"
- Run: `npm install`
- Then: `npm run dev`

### Error: "Cannot find module"
- Delete `node_modules` folder
- Run: `npm install`
- Then: `npm run dev`
